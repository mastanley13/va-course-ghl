# Plain Language + Glossary (A–Z)

This appendix centralizes quick, non-jargon definitions for recurring HighLevel terms. Each item includes a reminder of where to find it in the platform plus a screenshot placeholder to capture during walkthroughs.

## A–Z glossary with screenshot cues
- **Automation log** — the run history for workflow steps (success/failure timestamps).
  - Screenshot: Workflow "Execution log" sidebar showing individual steps and statuses.
- **Booking calendar** — the scheduling widget that controls when someone can book time.
  - Screenshot: Calendar availability editor with date/time grid visible.
- **Contacts** — the CRM records holding people or business details, tags, and activity.
  - Screenshot: Contact profile with timeline and tags highlighted.
- **Dashboard widget** — a tile summarizing KPIs like appointments, conversations, or revenue.
  - Screenshot: Dashboard home with widget tiles and drill-down links highlighted.
- **Evidence** — screenshots/exports proving a lab step was completed and tested.
  - Screenshot: Example lab submission with annotated screenshot and timestamp.
- **Funnel step** — an individual page/step inside a funnel or website journey.
  - Screenshot: Sites/Funnels editor showing the step list and selected page.
- **GPT/AI action** — workflow step that uses AI to summarize, classify, or personalize.
  - Screenshot: Workflow canvas with an AI Decision Maker node opened.
- **Human handover** — the rule that stops a bot and assigns a user when a person requests it.
  - Screenshot: Conversation AI Flow Builder node configured for "Human handover".
- **Integration** — a connection to an external service (Stripe, Google, Outlook, ad networks).
  - Screenshot: Settings → Integrations page showing connected accounts.
- **Journey filter** — combined AND/OR filters that refine inbox or dashboard drill-down results.
  - Screenshot: Conversations filter panel with multiple filters applied.
- **KPI** — key performance indicator tracked on the dashboard (bookings, revenue, replies).
  - Screenshot: Dashboard KPI tile expanded with detail view.
- **Labs toggle** — feature flag that enables in-development interfaces (e.g., new Conversations UI).
  - Screenshot: Settings → Labs tab with a toggle highlighted.
- **Membership area** — learning/content space for communities, lessons, and gated assets.
  - Screenshot: Memberships/Communities module showing a course outline.
- **Navigation breadcrumbs** — the header links showing where you are inside Sites, Workflows, or Assets.
  - Screenshot: Top-left breadcrumb trail inside the Sites builder.
- **Opportunity card** — the draggable deal card inside a pipeline stage with value/owner/status.
  - Screenshot: Opportunities board with an open card sidebar.
- **Pipeline stage** — a column in a pipeline representing a required action or outcome.
  - Screenshot: Pipeline view with stage names and WIP limits annotated.
- **Quiet hours** — messaging window that prevents calls/SMS from sending during off-hours.
  - Screenshot: Workflow settings tab showing quiet hours configuration.
- **Reputation widget** — review display/request elements embedded on sites or emails.
  - Screenshot: Reputation dashboard with a widget preview panel.
- **Snapshot** — packaged set of assets/settings to clone into another sub-account.
  - Screenshot: Agency view showing Snapshot creation/export options.
- **Trigger link** — tracked URL that logs clicks and can start automations.
  - Screenshot: Marketing → Trigger Links list with one link details pane.
- **UTM parameters** — URL tags (utm_source, utm_medium, utm_campaign) used for attribution.
  - Screenshot: URL builder modal with UTM fields filled out.
- **Voicemail drop** — pre-recorded voicemail sent without ringing the phone normally.
  - Screenshot: Workflow action drawer showing voicemail configuration.
- **Workflow canvas** — the visual builder where triggers, actions, and conditions are connected.
  - Screenshot: Workflow canvas zoomed out with labeled nodes.
- **Cross-channel routing (X-channel)** — using rules to decide which channel (SMS/email/call) to use next.
  - Screenshot: Workflow branch choosing SMS vs email based on consent and engagement.
- **Yield buffer** — extra time inserted before/after appointments or messages to avoid collisions.
  - Screenshot: Calendar buffer settings showing minutes before/after slots.
- **Zapier/webhook bridge** — external automation that catches webhooks or sends data out of HighLevel.
  - Screenshot: Webhook configuration modal with endpoint URL and sample payload.

## Navigation Verbs Quick-Reference (EN / ES / PH)
Use these side-by-side cards to keep action verbs consistent across languages when guiding learners or clients.

| Verb | English (EN) | Español (ES) | Filipino / Tagalog (PH) |
| --- | --- | --- | --- |
| Click | Click the **[button/link]** | Haz clic en **[botón/enlace]** | I-click ang **[button/link]** |
| Toggle | Toggle **[setting]** on/off | Activa o desactiva **[configuración]** | I-toggle ang **[setting]** papunta sa on/off |
| Save | Save your changes | Guarda tus cambios | I-save ang mga pagbabago |
| Publish | Publish the page/workflow | Publica la página/el flujo | I-publish ang page/workflow |
| Reconnect | Reconnect the integration/account | Vuelve a conectar la integración/cuenta | I-reconnect ang integration/account |

**How to use:** copy the phrasing directly into module steps, quiz explanations, and labs so learners see the same verb in EN/ES/PH side by side.
