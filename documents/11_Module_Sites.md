Sites is where the conversion happens: landing pages, booking pages, proposal pages, and form/survey capture. For capstone: an advertiser offer page + optional PDF download + booking page and/or payment step.

## Learning outcomes

- Deploy chat widget for live/AI chat where appropriate.

## Vocabulary
- **Funnel**: step-based pages, often conversion flow; supports A/B testing
- **Website**: multi-page site; includes store options (if enabled)
- **Page builder**: section/row/column/element structure
- **Form**: lead capture
- **Survey**: multi-step intake/qualifier; conditional logic where supported
- **Chat widget**: website chat entry point into Conversations
